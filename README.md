🔨 Mace Control: Maces can be completely disabled. When disabled: ❌ Pick up → deleted ❌ Inventory → deleted ❌ Use → deleted ❌ Attack → deleted

💪 Strength Limit

You can set the maximum Strength level:

/setstrength 1 Anything above this level will be: ❌ blocked ❌ unusable ❌ automatically removed

👉 Prevents overpowered potions in PvP

☠️ Weakness Control

Weakness can be completely disabled:

/toggleweakness When disabled: ❌ Potions will not work ❌ Effect will be removed
